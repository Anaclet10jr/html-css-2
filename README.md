# Mpuza Login Page

A modern and responsive **login page UI** built using **HTML and CSS**.  
This project focuses on clean design, usability, and professional layout.

---

## Features

- Clean and modern login interface  
- Email and password inputs with **icons inside fields**  
- Password field with **eye icon** (UI ready for show/hide)  
- “Remember Me” checkbox  
- Responsive design (works on desktop & mobile)  
- Structured and refactored CSS (no duplicates)

---

## Technologies Used

- **HTML5**
- **CSS3**
- Icons via image assets (PNG)
- for login input i. key icon ii. icon of eyes for password 3.
